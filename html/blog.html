May 19 - May 25 2025

This week I joined Progressive Insurance's Cloud Developver Enablement team. On this team I will work as a full stack developer working on an app called Cloudflow. This app is an API which enables Progressive employees to get involved in cloud through AWS or Azure account creation and management and infrastructure as code through Hashicorp Terraform. The API backend is written in python using flask to create API endpoints. The frontend is served through Retool. And the server infrastructure is managed through Terraform. During my first week I worked on gaining authroziation and setting up my developer environment. On Friday May 25th, I submitted my first pull request decomissioning an old endpoint.
